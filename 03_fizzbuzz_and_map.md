# Reduce Problem
``` javascript
let arr = ["hi","my","name","is","Ammar"]
function joinStrings(acc, word){
  console.log(acc)
  return acc + " " + word;
}
let newarr = arr.reduce(joinStrings, "");
console.log(newarr)
```
